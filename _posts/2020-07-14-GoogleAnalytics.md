## gitについての覚え書き
***
よくgitの仕方を忘れちゃうので書いておきます<br>

1.ファイルを指定する

```
cd <ファイル名>
```

2.すべてgitに加える

```
git add .
```

3.commitする

```
git commit -m "コメント"
```

4.pushする

```
git push origin master
```
